# analisi-sistema-sanitario
Analisi del sistema ospedaliero italiano tramite Power BI

Questo progetto di data analysis analizza il funzionamento del sistema ospedaliero italiano attraverso l’integrazione e l’elaborazione di dati open source relativi alle strutture sanitarie pubbliche.

L’obiettivo principale è valutare il livello di pressione del sistema sanitario e l’utilizzo delle risorse disponibili, individuando eventuali criticità territoriali e supportando processi di pianificazione e decision making in ambito sanitario.

Obiettivi dell’analisi

Analizzare la pressione ospedaliera a livello regionale

Valutare il tasso di utilizzo dei posti letto

Studiare la distribuzione delle dimissioni ospedaliere

Identificare aree con sovraccarico del sistema sanitario

Supportare l’individuazione di priorità di intervento

KPI (Key Performance Indicators)

I principali indicatori sviluppati nel progetto includono:

Totale dimissioni

Totale giornate di degenza

Numero di posti letto disponibili

Tasso di utilizzo dei posti letto (%)

Indice di pressione ospedaliera (%)

Durata media della degenza

Dataset

I dati utilizzati provengono da fonti open data ufficiali (es. Ministero della Salute) e includono:

Anagrafica delle strutture sanitarie

Attività ospedaliera (dimissioni, degenze)

Capacità delle strutture (posti letto)

Le informazioni sono state preprocessate per garantire qualità, coerenza e integrabilità.

Data Preparation

Le principali attività di preparazione dati includono:

Data cleaning (gestione missing values e outlier)

Standardizzazione delle variabili

Controllo di qualità e coerenza del dataset

Integrazione di fonti dati eterogenee

Costruzione di modelli dati relazionali.

Analisi e Metodologia

L’analisi è stata sviluppata attraverso:

Analisi esplorativa dei dati (EDA)

Costruzione di indicatori sintetici

Analisi territoriale a livello regionale

Confronto tra regioni (ranking e distribuzione)

Identificazione di pattern di sovraccarico.

Dashboard Power BI

È stata sviluppata una dashboard interattiva in Power BI che consente di:

Monitorare in tempo reale gli indicatori sanitari

Analizzare la distribuzione territoriale delle criticità

Esplorare i dati per regione e disciplina

Supportare decisioni basate su evidenze.
